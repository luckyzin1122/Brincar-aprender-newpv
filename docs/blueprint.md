## Style Guidelines:

- Primary color: Soft pink (#FF6B9D) to evoke a welcoming and gentle feeling.
- Background color: Very light beige/cream (#FFF9F0) to create a soft, inviting backdrop.
- Accent color: Sky blue (#87CEEB) to provide a touch of freshness and visual contrast.
- Headings font: 'Poppins', sans-serif for a modern, bold look. Note: currently only Google Fonts are supported.
- Body font: 'Inter', sans-serif for readability. Note: currently only Google Fonts are supported.
- CTA font: 'Poppins', sans-serif, semi-bold to draw the users attention to take action. Note: currently only Google Fonts are supported.
- Clean and spacious design with cards featuring a smooth border-radius (12-16px).
- Use modern and minimalistic icons to enhance visual communication.
- Subtle micro-animations on scroll (fade-in, slide-up) to create an engaging user experience.
